# Name Of IT

## Getting started

### Installation

```bash
$ pnpm install
```

### Copy `.env.example` to `.env`

```bash
$ cp .env.example .env
```

### Run the project

```bash
$ pnpm start
```

## Commands

### Build the project

```bash
$ pnpm build
```

### Run the project

```bash
# Run the project in development mode
$ pnpm start

# Run the project in watch mode
$ pnpm start:watch

# Run the project in debug mode
$ pnpm start:debug
```

### Run the tests

```bash
# Run the tests
$ pnpm test

# Run the tests in watch mode
$ pnpm test:watch

# Run the tests in coverage mode
$ pnpm test:coverage
```

Enjoy!
